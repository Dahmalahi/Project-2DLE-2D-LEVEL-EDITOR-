# Project-2DLE-2D-LEVEL-EDITOR-
2DLE is a complete 2D RPG engine for J2ME (MIDP 2.0) mobile devices. It includes a full tile-based level editor, a scripting language (2DLS), a battle system, NPC management, weather effects, day/night cycles, map transitions, and a distribution format (.2dip) that packages entire games for public release.
